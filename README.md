# Financial Control

Personal finance management system built with Spring Boot.

## Features
- Category and subcategory management
- Credit card installment purchases
- Monthly invoice generation
- Expense and income tracking

---

# Controle Financeiro

Sistema de controle financeiro pessoal desenvolvido com Spring Boot.

## Funcionalidades
- Cadastro de categorias e subcategorias
- Compras parceladas no cartão
- Geração de faturas mensais
- Controle de despesas e receitas
